# Book Tracker (Трекер прочитанных книг)

## Автор
**Гусев Артём**  

## Описание программы
Book Tracker — это десктопное GUI-приложение, разработанное на Python с использованием библиотеки Tkinter. Приложение позволяет пользователю вести учет прочитанных книг, сохранять данные в локальный файл и осуществлять быстрый поиск по коллекции.

### Основные возможности:
- Добавление книг с указанием названия, автора, жанра и количества страниц.
- Валидация ввода (проверка на пустые поля и корректность числа страниц).
- Просмотр списка книг в виде таблицы.
- Фильтрация книг по жанру и минимальному количеству страниц.
- Сохранение и загрузка базы данных в формате JSON.

## Требования
- Python 3.6 или выше
- Библиотека `tkinter` (обычно входит в стандартную установку Python)
